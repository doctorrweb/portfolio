# doctorrWeb Website

## Usage

Rename the config-template.js to config.js and fill it with the correct values

## Install Dependencies

````
npm install
````

> Frontend for doctorrWeb website


## Generate the bundle.js file
`````
npm run dev 
`````


> Backend API for doctorrWeb website


## Run App
`````
npm start
`````
